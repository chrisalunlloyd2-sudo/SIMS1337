## Data Flow
```
User Click → Button Handler → Scene/VBox Swap → UI Update
     ↓
Log Message → Console → Debug Trail
     ↓
Station Trigger → Command Queue → Terminal Execution
     ↓
Output → Next Model Input → Chain Processing
```

┌─────────────────────────────────────────────────────────┐
│                    MAIN SCENE                            │
